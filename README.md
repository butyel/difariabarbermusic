# DiFaria Barber Music — site institucional

Site em Next.js com foco em SEO local, conversão para WhatsApp e identidade visual verde-musgo, creme, branco e preto.

## Rodar localmente

```bash
npm install
npm run dev
```

## Antes de publicar

1. Troque `https://difariabarbermusic.com.br` pelo domínio definitivo em `app/layout.tsx` e `app/page.tsx`.
2. Confirme telefone, horários e lista real de serviços.
3. Substitua ou complemente as fotos em `public/images`.
4. Configure Google Search Console, Google Analytics e vincule o site ao Perfil da Empresa no Google.
5. Publique na Vercel, Cloudflare Pages ou servidor Node compatível.
